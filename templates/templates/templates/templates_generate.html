STEP 01

Index Block

The first thing to do is to create a page title and page description for the home or index page of your website. The index title will be used as the dynamic meta title, while the index description will be the meta description of your index page.

STEP 02

Value Block

After all, your business provides some sort of values to your customers, right? Using the variables from your CSV file, you can create dynamic value elements to attract more customers. It's ideal to create twelve, eight, or four values that will appear on all your topic and product landing pages.

STEP 03

Topic Block

Next, a topic is a categorical landing page with several dynamic topic elements that can be created using the variables from your CSV file. You need five topics i.e. topic-01 is the parent category of topic-02. The topic title will be used as the dynamic meta title, while the topic description will be the meta description of your topic page.

STEP 04

Product Block

Similarly, a product is a hierarchical landing page with several dynamic product elements that can be created using the variables from your CSV file. A product is created inside the last category, in order word, topic-05. The product title will be used as the dynamic meta title, while the product description will be the meta description of your product page.

STEP 05

Package Block

Almost done, the package section will appear on your topic and product landing pages that will quickly give your customers an idea about the pricing packages of your products and services. You can use the variables from CSV file.

STEP 06

FAQ Block

In addition, the FAQ section will be shown across your topic and product landing pages. This allows you to add dynamic customer questions. It's ideal to add up to five FAQs and you can use the same variables as above.
